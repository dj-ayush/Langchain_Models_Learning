**LangChain Models Learning** is a personal learning repository where I explore different LangChain models and APIs through simple Python examples.

The goal of this repository is to understand how different **LLMs, Chat Models, and Embedding Models** work using LangChain.

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/dj-ayush/Langchain_Models_Learning.git
cd Langchain_Models_Learning
```

### Create a virtual environment

```bash
python -m venv venv
```

Windows

```bash
venv\Scripts\activate
```

macOS / Linux

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Examples

Navigate to any folder and run the desired Python file.

Example:

```bash
python 1.LLMs/1_llm_demo.py
```
